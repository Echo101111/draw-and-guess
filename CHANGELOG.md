# Changelog

## 2.9.0 — 2026-05-27

- Feat: 新增反馈建议和管理后台，首页可提交反馈
- Feat: TURN 服务器（coturn）部署，跨网络语音通过热点/防火墙
- Feat: 语音面板精简为麦克风开关 + 聋音 + 退出，去掉发言按钮
- Fix: 加入语音默认静音，需手动点麦克风开麦（防社死）
- Fix: WebRTC P2P 连接竞态修复（双方同时发 Offer 导致概率性失败）
- Fix: 退出页面后彻底清理语音连接，防止持续通话
- Fix: 描述阶段轮到你了自动取消静音
- Fix: 描述轮换后上一位发言人光环不消失
- Fix: 语音描述无需打字，点"完成描述"即可切到下一位
- Fix: 选定词汇未参与随机选词
- Fix: AudioContext 被浏览器暂停后自动恢复
- Fix: ICE 连接失败后 3 秒自动重试

- Feat: 自动加载 .env 配置文件，简化本地开发环境变量管理
- Fix: 修复谁是卧底断线后发言错乱和状态重置
- Fix: 安全加固、稳定性修复与代码质量改进
