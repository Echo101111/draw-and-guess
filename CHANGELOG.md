# Changelog

## 2.3.0 — 2026-05-17

- Feat: 聊天框中直接猜词，猜对猜错即时反馈，不再需要单独的答题输入框
- Feat: 新增国内部署支持（Docker 镜像加速、部署脚本），一键部署到云服务器
- Fix: 大幅提升游戏稳定性，修复重连循环导致僵尸房间的问题
- Fix: 修复断线重连后开始游戏按钮无响应的问题
- Fix: 修复观战者下一轮无法正常参与游戏的问题
- Fix: 修复 Cloudflare 代理下 Socket.io 握手超时的问题
- Fix: 修复画面不同步和笔画渲染闪烁的问题
- Fix: 修复画布容器边框影响视觉的问题
- Test: 新增 Playwright E2E 测试框架及 3 个测试场景

- Perf: 画板响应更流畅，复杂笔画不再延迟卡顿
- Perf: 网络断开后自动重连更持久，大幅减少"连接失败"提示
- Perf: 进入房间更快，首页已预连接无需额外等待
- Fix: 轻触/点击画板可留下点状笔迹，方便点装饰
- Fix: 修复撤销后画师和猜题人画板不一致的问题
- Fix: 桌面端和移动端画板保持4:3等比例，画面不再拉伸变形
