# Changelog

## 2.5.0 — 2026-05-19

- Feat: 首页新增"贡献词库"入口，用户可以批量提交词语永久加入系统词库
- Feat: 新增敏感词过滤机制，防止不适当内容进入词库
- Feat: 词库新增角色（celebrities）和职业（professions）独立分类
- Feat: 新增同义词（synonyms）支持，猜词匹配更灵活
- Feat: 优化词条可画性评分（drawability），提升绘画体验
- Fix: 修复自定义词提交后需重启服务端才生效的问题——提交即生效
- Fix: 修复画布卸载时笔画缓存未清理导致的内存泄漏
- Fix: 修复聊天消息超过 500 条不自动截断导致的内存膨胀
- Fix: 修复游戏结束时画师选词状态和撤销缓存未清理的问题
- Fix: 修复 `removePlayerTimestamps` 未清除 keyed 条目的遗漏

## 2.4.0 — 2026-05-17

- Feat: 画师可从多个词语中选取本轮答案，更有参与感
- Feat: 选词超时（30秒）自动随机分配，游戏不卡住
- Refactor: 游戏核心逻辑大幅重构，为后续功能扩展打好基础
- Fix: 自定义词库房间中房主不能猜题的限制

## 2.3.0 — 2026-05-17

- Feat: 聊天框中直接猜词，猜对猜错即时反馈，不再需要单独的答题输入框
- Feat: 新增国内部署支持（Docker 镜像加速、部署脚本），一键部署到云服务器
- Fix: 大幅提升游戏稳定性，修复重连循环导致僵尸房间的问题
- Fix: 修复断线重连后开始游戏按钮无响应的问题
- Fix: 修复观战者下一轮无法正常参与游戏的问题
- Fix: 修复 Cloudflare 代理下 Socket.io 握手超时的问题
- Fix: 修复画面不同步和笔画渲染闪烁的问题
- Fix: 修复画布容器边框影响视觉的问题
- Test: 新增 Playwright E2E 测试框架及 3 个测试场景

## 2.2.0 — 2026-05-16

- Feat: 画师可使用撤销功能（Ctrl+Z / 工具栏 ↩️ 按钮），逐笔撤销同步到所有人
- Perf: 画板响应更流畅，复杂笔画不再延迟卡顿
- Perf: 网络断开后自动重连更持久，大幅减少"连接失败"提示
- Perf: 进入房间更快，首页已预连接无需额外等待
- Fix: 轻触/点击画板可留下点状笔迹，方便点装饰
- Fix: 修复撤销后画师和猜题人画板不一致的问题
- Fix: 桌面端和移动端画板保持4:3等比例，画面不再拉伸变形
